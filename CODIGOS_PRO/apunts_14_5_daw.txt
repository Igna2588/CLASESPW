1. Scope
	Scope de clase -> todo aquello declarado en la clase se puede utilizar en cualquier parte
	Scope de metodo / bloque -> todo aquello declarado dentro de un bloque, solo se puede utilizar dentro de dicho bloque

2. final
	la variable no puede cambiar de valor
	final array[] a = new array[7] -> array[1] = 78
	final Arraylist a = new ArrayList() -> arraylist.add(123) arraylist.remove(123)
	fina Usuario u = new Usuario -> u.setNombre("Nuevo") u.setApellido("asdasd")
	final int dato = 5
	No se puede
		a = new int[6]
		a = new Arraylist()
		dato = 6
		u = new Usuario()

4. Modificadores acceso
	
	siempre y cuando tenga una instancia de la clase (objecto)

	public -> aquellos que dan acceso desde cualquier clase (sitio)
	protected -> aquellos que se pueden acceder desde la propia clase y desde las clases internas (heredadas) y del mismo paquete
	private -> aquellos que solo son accesibles desde la clase donde estan definidos -> setter / getter

9. Array

	tipo[] nombre = new tipo[] -> mal
	int[] nombre = new int[4] -> [0,0,0,0] -> bien porque se indica solo cantidad
	int[] nombre = new int[]{1,2,3,4} -> bien porque se indica solo valores
	int[] nombre = {1,2,3,4} -> bien -> bien porque se indica solo valores
	int[] nombre = new int[5]{1,2,3,4} -> {1,2,3,4,0} -> mal porque se indica cantidad y valores
	int[] nombre = new int[4]{1,2,3,4} -> {1,2,3,4} -> mal porque se indica cantidad y valores
	O defines un array indicando su capacidad o defines el array indicando sus elementos -> nunca las dos cosas al mismo tiempo
